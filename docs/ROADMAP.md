# Roadmap

## Phase 0 — Foundation

Safe local OS with Markdown memory, CSV inputs, report generation, tests, and CI.

## Phase 1 — Data snapshots

Add cached price snapshots, SEC metadata snapshots, and macro snapshots.

## Phase 2 — AI drafting

Add prompt templates that use saved context to draft reports without enabling trades.

## Phase 3 — Paper sandbox

Add paper-trading simulation only. It must be disabled by default and never share code paths with live trading.

## Phase 4 — Dashboard

Add a simple dashboard or notebook for weekly reviews.
