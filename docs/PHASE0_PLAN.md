# Phase 0 Plan

## Goal

Create a safe, local-first research OS that makes investment study repeatable.

## Deliverables

- Repo structure
- Policy memory files
- Routine prompts
- CSV loaders
- Data-provider URL helpers
- Report generator
- Scorecard
- CLI
- Tests
- CI

## Out of scope

- Live trading
- Brokerage APIs
- Automated execution
- Real-time alerting
- Portfolio optimization
