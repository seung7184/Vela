# Portfolio Rules

## Default allocation logic

1. Build and maintain the ETF core first.
2. Treat single stocks as research ideas, not default purchases.
3. Add single-stock exposure only when the thesis, valuation, and downside risk are clear.
4. Avoid overlapping exposures that only duplicate existing ETF risk.

## Decision labels

Allowed labels:

- Avoid
- Study more
- Watchlist
- Small position candidate
- ETF-only is better

## Review cadence

- Daily: scan only for material changes.
- Weekly: update thesis log and mistake log.
- Monthly: check whether the watchlist still deserves attention.
